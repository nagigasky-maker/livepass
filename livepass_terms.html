本利用規約（以下「本規約」）は、LIVE PASS（以下「本サービス」）の提供条件および本サービスの運営者（以下「当社」）と登録ユーザー（以下「ユーザー」）との間の権利義務関係を定めるものです。本サービスをご利用いただく前に、本規約の全文を必ずお読みください。

第1条（適用）

  1. 本規約は、本サービスの提供条件およびユーザーと当社との間の一切の関係に適用されます。
  2. 当社が本サービス上で掲載する個別規定、ガイドライン、ポリシーは本規約の一部を構成します。
  3. 本規約と個別規定が矛盾する場合、個別規定が優先します。

第2条（定義）

  • コンテンツ — ユーザーが本サービス上で投稿・送信するテキスト、画像、動画、音声、その他一切の情報
  • アーティスト — 楽曲・映像・パフォーマンス等を本サービス上で発信するユーザー
  • 主催者 — イベント・ワークショップ・展示を本サービス上で企画・運営するユーザー
  • PASS — 本サービスが提供するサブスクリプション型アクセス権

第3条（アカウント登録）

  1. ユーザーは正確かつ最新の情報を提供する義務を負います。
  2. 13歳未満の方は本サービスを利用できません。18歳未満の方は保護者の同意が必要です。
  3. アカウントおよびパスワードの管理責任はユーザーが負います。第三者による不正利用について当社は一切責任を負いません。
  4. 当社は、虚偽の登録、過去に規約違反のあった者、その他当社が不適当と判断した場合、登録を拒否または抹消することができます。

第4条（料金プラン）

  1. 本サービスは、FREE / PRO / BIZ の3プランを提供します。
  2. PRO（月額980円）およびBIZ（月額2,980円）は有料プランであり、決済はStripe社を通じて行われます。
  3. 有料プランは自動更新されます。解約は次回更新日の前日までに行ってください。
  4. 料金は事前にお支払いいただくものとし、原則として返金はいたしません。

第5条（収益分配）

  1. アーティスト・主催者は本サービスを通じてコンテンツの販売、チケット販売、サブスクリプション料金の受領等により収益を得ることができます。
  2. 決済はStripe Connect（または同等の決済代行事業者）を介して行われ、本人確認（KYC）・利用規約同意・税務情報の提出が必要となる場合があります。
  3. 当社は売上から所定のプラットフォーム手数料（別途料金表に定める）を控除した金額を、登録された銀行口座へ送金します。
  4. コラボレーター機能を用いた収益分配は、投稿時に設定された分配率に従って自動的に行われます。分配率の設定および合意はユーザーの自己責任です。

第6条（コンテンツの取扱い）

  1. ユーザーが本サービスに投稿したコンテンツの知的財産権は、原則としてユーザー本人に帰属します。
  2. ユーザーは当社に対し、当該コンテンツを本サービスの提供・宣伝・改善の目的で無償・非独占的に利用する権利を許諾するものとします。
  3. ユーザーは、投稿するコンテンツが第三者の権利（著作権・肖像権・商標権・プライバシー権等）を侵害しないことを保証します。
  4. ユーザーは自己のコンテンツについて、自らの責任において管理するものとします。

第7条（禁止事項）

ユーザーは以下の行為をしてはなりません。

  1. 法令または公序良俗に違反する行為
  2. 犯罪行為に関連する行為
  3. 第三者の知的財産権、肖像権、プライバシー、名誉を侵害する行為

  5. マネーロンダリングまたはそれに類する行為
  6. 未成年者に有害な情報を発信する行為
