# PASS Card Assets

Place final card artwork here.

## Naming convention

`{event-slug}-{venue}-{date}.{ext}`

Examples:
- `amd-clubasia-2026-05-04.jpg` — ALL MUST DANCE, Club Asia, May 4, 2026
- `sc-unit-2025-03-21.jpg` — SPACE COOKING, Unit, Mar 21, 2025

## Referenced in

- `livepass_flashback.html` — main card reveal
- `livepass_collection.html` — PASS HOLDER fan

## Recommended specs

- Portrait orientation
- Min 1200×1800 (2x retina)
- JPEG or PNG
- File size under 500KB each (use mozjpeg / squoosh)
